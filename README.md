## ⚙️ Funcionalidades

- Conversão de tipos (datas, valores monetários)
- Preenchimento de valores ausentes com defaults
- Eliminação de duplicatas
- Criação de coluna `Total_Vendas`
- Exportação final em `.csv` com os dados limpos
